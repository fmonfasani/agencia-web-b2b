✨ ANIMACIONES
==============

Page Load:
  • Skeleton screens en lugar de spinners
  • Fade-in en 200ms
  
Card Entrance:
  • Stagger children (100ms delay entre cards)
  • Slide-up + fade-in
  
Hover Effects:
  • Button: background change (150ms)
  • Card: translateY(-8px) + enhanced shadow
  • Image: zoom(1.05) + overlay
  
Rating Stars:
  • Click con spring animation
  • Tooltip en hover
  
KPI Numbers:
  • Count-up animation (0 → target number)
  • Duration: 1 segundo
  
Toast Notifications:
  • Position: top-right
  • Colors: green (success), red (error), blue (info)
  • Auto-dismiss: 5 segundos
  • Slide-in from right

🛠️ STACK RECOMENDADO

Charts:
  • Recharts (React charts library)
  • Chart.js como alternativa

Tables:
  • TanStack Table (React Table)
  • DataGrid de Material-UI

Forms:
  • React Hook Form
  • Zod o Yup para validación

Animations:
  • Framer Motion (recomendado)
  • React Spring como alternativa
